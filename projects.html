DIY TV Ambient Lighting System

Iterative Personal Project (2019 - Present)

Developed a custom, low-latency TV ambient lighting system using addressable LEDs (WS2812B). The current iteration runs on an ESP32 with WLED firmware, receiving screen data wirelessly from HyperHDR. Features include sound reactivity and full integration with Home Assistant for automation. Initial prototypes explored Raspberry Pi and basic transistor circuits.
