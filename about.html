About Me

Hello! I'm Satyam Kumar, a Computer Science graduate from Goldsmiths, University of London. While my academic background provides a strong foundation, my true passion lies at the intersection of software, hardware, and creative problem-solving – a journey sparked by the electrifying visuals of EDM concerts like Tomorrowland and Ultra Miami.

This inspiration led to projects like my long-term DIY TV Ambilight system, evolving from simple LEDs to a complex setup integrated with WLED, HyperHDR, and Home Assistant. It's this hands-on approach – tinkering, building, breaking, and learning – that truly drives me.

Resourcefulness & Problem Solving

As a student, often working with limited resources, I developed a knack for finding creative and cost-effective solutions. This mindset is core to how I approach technology:

  • Building a NAS for (Almost) Free: I constructed a functional TrueNAS Core server by salvaging hard drives from discarded laptop, non-working external drive, and an old DVR. Combining mirroring for redundancy (2x old HDDs) and striping for speed (SSD + DVR HDD), I created useful storage pools without significant upfront cost, running it within a VirtualBox VM on my PC.
  • Cost-Effective Home Automation Cloud Access: Instead of opting for the paid Home Assistant Cloud subscription, I researched and implemented a free alternative using Cloudflare Zero Trust tunnels and Google Assistant API integrations to achieve secure remote access and voice control.

This drive to find workarounds, learn from forums and communities, and make things work translates directly into my development approach – seeking efficient, practical, and sometimes unconventional solutions to challenges.

Hands-On Exploration & Projects

My curiosity extends across various tech domains, manifesting in practical projects:

  • Evolving IoT & Home Automation: Beyond Ambilight, I run a comprehensive Home Assistant setup (migrated from a VM to a Raspberry Pi 4B) integrating various devices using ESPHome, Google Assistant, and secure remote access via Cloudflare. I even use HASS Agent for Windows integration.
  • Network Attached Storage (NAS): The aforementioned TrueNAS Core build from salvaged parts, demonstrating practical knowledge of storage concepts (Mirroring/RAID 1, Striping/RAID 0), virtualization (VirtualBox), and hardware sourcing.
  • Exploring Local LLMs: I actively experiment with running Large Language Models like Deepseek, Gemma, Code Llama, and Qwen locally on my hardware. I'm currently exploring ways to integrate these LLMs into my Home Assistant instance for advanced automation possibilities.
  • Early Projects: My journey included foundational projects like the cloud-based light switch and the Merkle rex crypto simulator, which built essential coding and problem-solving skills.
